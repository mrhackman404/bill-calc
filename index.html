How to Calculate Electricity Bill?

  1. Get the current meter reading in units
  2. Subtract current meter reading from previous month's total reading
  3. Select the consumer type from the dropdown (Example: For home electricity, select Residential LT-A)
  4. Enter the Unit and Load values. Check your bill paper for Load value
  5. Click on Calculate Bill
